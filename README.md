Portfilter
===========

Very Lightweight Portfolio Filter for Bootstrap 3


Usage
-----

	For handlers
		<tag data-toggle="portfilter" data-target="targetname">...
	For items
		<tag data-tag="targetname">...
Requisites
----------
	jQuery (http://jquery.com/)
